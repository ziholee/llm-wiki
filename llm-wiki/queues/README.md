# Queues

This folder is generated by the role-based dispatch workflow.

Role definitions are available in both English and Korean under `llm-wiki/roles/`.

## Files

- `dispatch-report.md`: overall routing summary
- `collector.md`: items waiting for source review and ingest
- `synthesizer.md`: items ready for synthesis into wiki pages
- `linker.md`: items that should be connected to related pages
- `critic.md`: items that need uncertainty and conflict review
- `publisher.md`: items that should be polished into stable summaries

Run `make wiki-dispatch` again whenever inbox contents change.

Recommended reading order for first-time users:

1. `dispatch-report.md`
2. `collector.md`
3. `synthesizer.md`
4. `linker.md`
5. `critic.md`
6. `publisher.md`

How to use the queue files:

- treat each `Recommended action` line as the next task to perform
- move from one role file to the next only after the previous role's work is reflected in `llm-wiki/`
